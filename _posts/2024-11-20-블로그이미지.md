## 좌측 베너
* 좌측 베너
`_sass/components/sidebar.scss`의 경로로 이동


```css
#sidebar {
  @include pl-pr(0);

  position: fixed;
  top: 0;
  left: 0;
  height: 100%;
  overflow-y: auto;
  width: $sidebar-width;
  background: var(--sidebar-bg);
  border-right: 1px solid var(--sidebar-border-color);

  /* Hide scrollbar for IE, Edge and Firefox */
  -ms-overflow-style: none; /* IE and Edge */
  scrollbar-width: none; /* Firefox */

  /* Hide scrollbar for Chrome, Safari and Opera */
  &::-webkit-scrollbar {
    display: none;
  }
```
* 이후 의 부분을 삭제한 후

```css
#sidebar {
  @include pl-pr(0);

  position: fixed;
  top: 0;
  left: 0;
  height: 100%;
  overflow-y: auto;
  width: $sidebar-width;

  /* 기존 배경 속성 변경 */
  background: var(--sidebar-bg); /* 기존 색상 */
  background-image: url('https://github.com/user-attachments/assets/a7808285-907a-4cdc-a151-dcbc6a747f60'); /* 배경 이미지 */
  background-size: cover; /* 배경 크기 */
  background-position: center; /* 배경 위치 */
  background-repeat: no-repeat; /* 반복 금지 */

  border-right: 1px solid var(--sidebar-border-color);

  /* 스크롤바 숨기기 - 기존 유지 */
  -ms-overflow-style: none; /* IE와 Edge */
  scrollbar-width: none; /* Firefox */
  &::-webkit-scrollbar {
    display: none; /* Chrome, Safari 등 */
  }
}
```
* 위 코드로 삽입
